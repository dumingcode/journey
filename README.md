# journey
记录教育培养之路
## 2019
本年度英语学习目标是进行启蒙教育，从super simple song开始带着孩子唱歌。  
逐步开始播放0-3岁低幼原版英文动画片，充分发挥孩子母亲英语专业背景，在生活中多对孩子说英语。  
动词配合肢体语言、名词指着对应物体说。  
我主要负责查找各种网络资源，带着孩子看英文动画片，每天20-30分钟。
